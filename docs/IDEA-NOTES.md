# Idea Notes

## Current Best Framing

Review See-Through: turning AI reviews into transparent evidence layers.

## One-Liner

Review See-Through decomposes an AI-generated peer review into grounded criticism, weak criticism, off-scope criticism, hidden assumptions, missing evidence, and the next runnable Ralph Loop experiment.

## Why It Is Better Than Generic Claim Review

Generic claim review asks:

> Is the paper true?

Review See-Through asks:

> Is the review itself useful, grounded, and actionable enough to drive the next loop?

That is more Ralphthon-native because the review is not the end product. It is fuel for the next iteration.

## See-Through Analogy

| Image See-Through | Review See-Through |
|---|---|
| visible illustration | review prose |
| semantic layers | criticism layers |
| depth order | evidence depth |
| occluded body parts | hidden assumptions |
| inpainted missing regions | missing experiments |
| PSD output | evidence ledger + experiment card |

## Demo Pitch

The demo should show a review that contains one useful criticism, one vague criticism, and one bad criticism. Review See-Through separates them and prevents the next Ralph Loop from wasting time on the bad one.
