# Reviewer Disagreement Synthesizer

## Claims

1. The synthesizer clusters reviewer disagreements into 7 issue groups on 18 synthetic reviews.
2. It reports inter-annotator agreement with Cohen kappa of 0.71.
3. No real program committee discussion logs are used.

## Evidence

The cluster table contains 7 issue groups from 18 synthetic reviews. The agreement section reports Cohen kappa of 0.71. The data statement says no real program committee discussion logs are used.
