# Agent Prompt Compression

## Claims

1. The compressed prompt reduces average prompt tokens from 2400 to 900.
2. The compressed prompt preserves task success on the local 20-task benchmark.
3. The compressed prompt has not been tested on multi-file refactors.

## Evidence

The benchmark table reports 18/20 success for the full prompt and 18/20 success for the compressed prompt. The token count table reports 2400 tokens before compression and 900 after compression. Multi-file refactors are listed as future work.
